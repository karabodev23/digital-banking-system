# 💳 Digital Banking System

A full-stack banking simulation system built using ASP.NET Core and React, demonstrating real-world financial operations and clean architecture.

---

## 🚀 Features

- Create bank accounts
- Deposit funds
- Withdraw funds with validation
- Transfer money between accounts (atomic transactions)
- View transaction history
- Modern React dashboard UI

---

## 🧠 Key Concepts

- Clean Architecture (API, Application, Domain, Infrastructure)
- Entity Framework Core
- SQL Server integration
- ACID transactions
- RESTful API design
- Frontend-backend integration (React + .NET)

---

## 🛠 Tech Stack

### Backend
- ASP.NET Core Web API
- C#
- Entity Framework Core
- SQL Server

### Frontend
- React.js
- JavaScript
- CSS (Glassmorphism UI)
